# AppCompass Notifier

## Description

A microservice for the purpose of sending emails, slack notifications, sms, or any other external notification service integrations for AppCompass Services.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
